# Jacque Jeme - Interactive Design Studio

An immersive, space-themed web design platform for showcasing fashion and designs with interactive 3D-like models and real-time body measurement adjustments.

## Features

- **Flying Through Space Effect**: Animated starfield background creating a cosmic atmosphere
- **Interactive Human Models**: Switch between female and male body types
- **Real-Time Measurements**: Adjust body proportions and see instant visual feedback
  - Height (150-200 cm)
  - Shoulders (30-60 cm)
  - Chest (70-130 cm)
  - Waist (50-110 cm)
  - Hips (70-130 cm)
- **Color Gradient System**: Customize outfit colors with two-color gradients
- **Texture Options**: Apply different patterns (solid, striped, dotted, geometric)
- **File Upload**: Import custom designs and textures
- **Brand Colors**:
  - AI Midnight: #00474F (Deep Rich Teal)
  - AI Ice Crystal: #C2F0F7 (Icy Blue)

## How to Use

1. Open `index.html` in a modern web browser
2. Select gender (Female/Male)
3. Adjust body measurements using the sliders
4. Customize outfit colors using the color pickers
5. Apply gradients with the "Apply Gradient" button
6. Choose textures from the dropdown
7. Upload custom designs if desired
8. Press "Reset to Defaults" to start over

## Keyboard Shortcuts

- **R**: Reset all settings
- **M**: Switch to male model
- **F**: Switch to female model
